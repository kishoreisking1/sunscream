# Cosmetics Website

A modern and responsive Cosmetics Website built using React.js. The application provides a clean user interface with multiple pages and client-side routing for seamless navigation.

## Features

* Responsive design
* React Router navigation
* Home page
* Menu page
* About page
* Modern UI layout
* Reusable React components

## Technologies Used

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3
* React Router DOM

## Installation

1. Clone the repository

```bash
https://github.com/kishoreisking1/sunscream.git
```

2. Navigate to the project folder

```bash
cd cosmetics
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm start
```

The application will run at:

```text
http://localhost:3000
```
